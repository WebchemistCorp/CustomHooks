# React Custom-Hooks

## List

- useInput
- useQueryString
- use

## Install

```bash
npm install --save custom-hooks
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'custom-hooks'
import 'custom-hooks/dist/index.css'

const Example: React.FC = () => {
  return <MyComponent />
}
```
